# Assignment 5 Part 2 - Playwright UI Testing

Automated UI tests for the DePaul University bookstore using Playwright and JUnit 5.

## Test Cases
- TestCase 1: Search, filter, and add JBL earbuds to cart
- TestCase 2: Verify shopping cart details and proceed to checkout
- TestCase 3: Create Account / Proceed as Guest
- TestCase 4: Contact Information
- TestCase 5: Pickup Information
- TestCase 6: Payment Information
- TestCase 7: Delete product and verify empty cart

## How to Run
```
mvn test
```
